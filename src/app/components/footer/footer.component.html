Multicore Project
The project consists in the development of one application using multicore/multi-thread (first parallelism level), that makes a "Sales" scrapping of computer parts or similars that are offered in Amazon.com and NewEgg.com. The app also makes another scrapping from other websites (second parallelism level) to get benchmark information about some popular computer components as Graphic Cards(GPU) and Processors(CPU), on the server level the app extracts this data using multicore and then returns data. Finally the whole system has a front-end web app actually hosted in a web page, where all users can check the data got from scrapping this websites. All Sales and benchmark information is shown in this front page. Server level app also has all of this requests with and without parallelism.
